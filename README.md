# 📊 Tata GenAI Delinquency Risk Prediction

## 🚀 Project Overview

This project was developed as part of the **Tata Group's GenAI-Powered Data Analytics Virtual Experience Program**.

The objective of this project is to analyze customer financial behavior and design an AI-driven framework capable of identifying customers at risk of becoming delinquent. The project combines Exploratory Data Analysis (EDA), predictive modeling, and responsible AI principles to support proactive financial risk management.

---

## 🎯 Business Problem

Financial institutions face significant challenges in identifying customers who are likely to miss future payments.

The goal of this project is to:

* Identify key factors contributing to customer delinquency.
* Generate business insights through Exploratory Data Analysis (EDA).
* Develop an explainable machine learning model to predict delinquency risk.
* Support data-driven decision-making for collection and intervention strategies.

## 📈 Exploratory Data Analysis (EDA)

The EDA phase focused on:

* Data quality assessment and missing value analysis
* Distribution analysis of financial variables
* Correlation analysis using heatmaps
* Identification of delinquency risk indicators
* Customer behavior analysis

### Key Insights

* Credit Utilization emerged as a significant risk factor.
* Income and Debt-to-Income Ratio showed strong relationships with customer risk profiles.
* Historical payment behavior played an important role in delinquency prediction.

---

## 🤖 Predictive Modeling

### Selected Model

**Logistic Regression**

The model was selected because:

* It is highly interpretable.
* It is suitable for binary classification problems.
* It supports transparent and explainable decision-making.
* It aligns with responsible AI practices in financial services.

### Modeling Workflow

1. Data preprocessing
2. Missing value imputation
3. Categorical feature encoding
4. Feature scaling
5. Train-test split
6. Model training
7. Model evaluation
8. Performance interpretation

---

## 📊 Evaluation Metrics

The model was evaluated using:

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC Score
* Confusion Matrix

Special emphasis was placed on Recall and F1 Score to ensure effective identification of high-risk customers.
